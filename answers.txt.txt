1) Python programs to generate te Fibonacci sequence.
	
	a).Using looping technique:

	def fibonacci(n):
		a,b = 1,1
		for i in range(n-1):
				a,b = b,a+b
		return a
	print fibonacci(5) // 5 is the input how many number we need.

	b).Using recursion

	def fibRecursion(n):
		if n==1 or n==2:
			return 1
		return fibonacci(n-1) + fibonacci(n-2)
	print fibRecursion(5)
